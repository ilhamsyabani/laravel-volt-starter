## [1.0.0] - 2026-XX-XX

### Added
- Initial release
- `volt-starter:install` command with `--auth`, `--roles`, `--showcase` flags
- `volt-starter:page` Volt + Folio page generator
- `volt-starter:crud` full CRUD generator
- `HasRoles` trait for User model
- Toast notification system
- Dark mode support
- Responsive sidebar layout
- 12 Tier 1 UI components (Button, Input, Textarea, Select, Checkbox, Radio, Toggle, Badge, Card, Alert, Toast, Spinner)
- CSS variable theme system (default, rose, emerald, amber, sky)
- 8 Tier 2 UI components (Modal, Dropdown, Table, Pagination, Tabs, Breadcrumb, Avatar, Tooltip)
- Skeleton & Empty State components
- Component showcase page at `/showcase`
