# Contributing to Laravel Volt Starter

Thank you for considering contributing! Here's how to get started.

## Setup

```bash
git clone https://github.com/ilhamsyabani/laravel-volt-starter
cd laravel-volt-starter
composer install
```

## Running Tests

```bash
./vendor/bin/pest
```

## Pull Request Guidelines

1. Fork the repository and create a feature branch: `git checkout -b feature/my-feature`
2. Write tests for any new functionality
3. Run `./vendor/bin/pint` to format your code
4. Submit a PR with a clear description of what you changed and why

## Reporting Issues

Please include:
- Laravel version
- Livewire/Volt version
- PHP version
- Steps to reproduce
- Expected vs actual behavior

Thank you! 🙏
